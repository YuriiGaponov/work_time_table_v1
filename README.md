# work_time_table
PET-проект. Одной из целей разработки является закрепление навыков написания парсеров, работы с форматами офисных документов, создания графического интерфейса.
## Назначение
Десктопное приложение для ведения табеля учета рабочего времени сотрудников организации. Общая концепция: приложение парсит производственный календарь, сохраняет в БД. Хранит список сотрудников организации, выводит результаты в виде табеля рабочего времени по месяцам в виде .xlsx или .docx.
## Технологии
- Python 3.13
- SQLAlchemy 2.0
- sqlite3
## Способ запуска
Предполагается запуск из консоли и простой GUI на базе Python.
## Состав
Приложение состоит из нескольких модулей, каждый из которых отвечает за определенный функционал:
-  **parser**
Парсит производственный календарь с сайта Консультант+
-  **db**
Управление базой данных
-  **core**
Общие настройки приложения
